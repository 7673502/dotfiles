# dotfiles
- Text editor: [neovim](nvim/)
- Terminal: [ghostty](ghostty/)

## Setup
- Install ghostty and neovim.
- Install the [JetBrainsMono Nerd Font](https://www.nerdfonts.com/font-downloads).
- I use GNU stow (if on MacOS, install with homebrew using `brew install stow`) to link these folders to `~/.config/` using `stow nvim` and `stow ghostty`.
